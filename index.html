6. Software & Operating Systems Depth

While hardware represents the physical body, software is the intellect. Without software, hardware cannot function.

System Software (Operating Systems)

The OS allocates memory, manages files, and interfaces with hardware devices.

  • Windows: The most common GUI-based OS for personal computers.
  • Linux/UNIX: Open-source, highly secure OS used predominantly in servers.
  • macOS & Android: Proprietary and mobile operating systems.

Application Software

End-user programs designed to accomplish specific tasks.

  • Word Processors: MS Word (create/edit documents).
  • Spreadsheets: MS Excel (data tabulation).
  • DBMS: MySQL, MS Access (manage databases).

Graphical User Interface & Multi-User Architecture

  • Window: A viewing area on a computer display screen. the Windows operating system had a graphical user interface and allows for multiple viewing areas.
  • Icon: A small picture that represents a file or a program. Clicking or double clicking on the icon will activate it.
  • Multi user: The CPU is avalable for more then one user at time. Data can be shared, Resources can be shared, Economical.

Computer Languages & Translators

  • Assembler: Converts low-level assembly language into machine code.
  • Compiler: Scans the entire source code and translates it into machine code all at once.
  • Interpreter: Translates source code into machine code line-by-line during execution.

9. Cybersecurity & Threat Mitigation

Cybersecurity involves practices to protect networks, computers, programs, and data from attack, damage, or unauthorized access.

Common Digital Threats

  • Virus:

  • Worm: Replicates and spreads across networks automatically.
  • Trojan Horse: Destructive software disguised as a legitimate application.
  • Ransomware: Malware that encrypts a victim's files for a ransom payment.
  • Phishing: Fraudulent practice of sending emails purporting to be from reputable companies.

Defensive Countermeasures

  • Antivirus / Anti-malware: Scans, identifies, isolates, and eliminates malicious software.
  • Firewall: Monitors and controls incoming and outgoing network traffic.
  • Encryption: Encoding information so only authorized parties can access it.
  • Authentication: Using strong passwords and MFA to verify identity.
